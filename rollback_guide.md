# 버전 1.0 원복 가이드

현재의 안정적인 상태는 깃 태그 `v1.0.0`으로 고정되어 있습니다. 향후 신규 작업 중 문제가 발생하여 현재 상태로 되돌리고 싶을 경우 아래 절차를 수행하면 됩니다.

## 원복 방법

1. 로컬 작업 내용 폐기 및 원복 (터미널에서 실행)
   - `git reset --hard v1.0.0` 명령어를 실행하면 현재 로컬의 모든 수정 사항이 삭제되고 버전 1.0 상태로 강제 복구됩니다.

2. 원복 상태 강제 배포
   - `git push -f origin main` 명령어를 실행하여 원복된 로컬 상태를 원격 저장소에 강제로 덮어씁니다. 이에 따라 Vercel에서도 버전 1.0이 다시 배포됩니다.

## 인지 사항
- 저에게 "버전 1.0으로 원복해줘"라고 명령하시면 제가 직접 위 과정을 수행하여 시스템을 현재 상태로 되돌리겠습니다.
- 모든 대화 기록은 history.txt에 보관되므로, 원복 시점의 맥락을 파악할 수 있습니다.
